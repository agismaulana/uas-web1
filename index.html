Cara Kerja Website

User Mengakses sebuah URL di browser, website tersebut akan mengembalikan sebuah halaman berupa gambar, text dan beberapa media grafis lainnya, melalui sebuah method bernama http yang dikirimkan dari sisi client dan akan di proses oleh sisi server, selain itu website terbagi terhadap dua bagian yaitu website dinamis dan website statis, website dinamis terbagi terhadap tiga bagian yaitu bagian awal, bagian navigasi, bagian konten dan bagian akhir, sedangkan website statis terbagi terhadap dua bagian yaitu bagian awal dan bagian konten

Hubungan Antara Web Server, Domain, dan Hosting

Hubungan antara web server, domain serta hosting merupakan sebuah fondasi dalam melakukan publikasi sebuah website, dimana web server merupakan sebuah hardware yang akan menjalankan sebuah website, domain merupakan alamat yang mendaftarkan website untuk dapat dipublikasi dan berupa alamat yang dapat mudah dibaca oleh user, hosting merupakan sebuah penyedia layanan tempat website disimpan dan diakses oleh user
